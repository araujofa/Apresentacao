Sobre o projeto / about the project

O projeto "Rolê Top" foi um projeto de avaliação do primeiro semestre do SENAI, cuja finalidade era avaliar
HTML, CSS e C# atraves com MVC. Nesse projeto nós implementamos um simples banco de dados composto,
basicamente, pelo metodo "ObterPor" e "PrepararRegistroCSV" que coleta as informações dadas pelo
usuario atraves de um formulario e joga em um arquivo .CSV no diretorio "Database".

The project "Role Top" was the project of avaliation at the first semester at SENAI, whose purpose it was
avaliate the HTML, CSS and C# with MVC. In this project we implemented a simple database composed, basically,
of an method called "ObterPor" and other called "PrepararRegistroCSV" wich take the informations gived
by the user through the form and create an .CSV archive on "Database" directory.
